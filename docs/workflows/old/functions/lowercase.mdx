### Input

A string.

### Output

Returns the string which is lowercased.

### Example

```yaml
actions:
  - name: trigger-slack
    condition:
      - type: equals
    value: keep.lowercase('ABC DEF')
    compare_to: "abc def"
    compare_type: eq
```
