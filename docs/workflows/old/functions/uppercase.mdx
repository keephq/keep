### Input

A string.

### Output

Returns the string which is uppercased.

### Example

```yaml
actions:
  - name: trigger-slack
    condition:
      - type: equals
    value: keep.uppercase('abc def')
    compare_to: "ABC DEF"
    compare_type: eq
```
