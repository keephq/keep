## Overview

Matrix Provider allows you to send alert messages from Keep via [Matrix](https://matrix.org/), an open protocol for decentralized communication. The Matrix provider supports sending plain text and HTML formatted messages to any Matrix room.

<AutoGeneratedSnippet />

## Connecting with Matrix Provider

1. Get your Matrix access token from your Element client (Settings -> Help & About -> Advanced -> Access Token).
2. Identify your homeserver URL (e.g. `https://matrix.org` or your self-hosted server).
3. Get the Room ID of the target room (Room Settings -> Advanced -> Internal room ID).
4. Add Matrix Provider in Keep with these credentials.

## Message Format Support

### Text Messages
Send plain text messages:

```yaml
with:
  message: "Alert: High CPU usage detected on server-01"
```

### Formatted Messages (HTML)
Send HTML formatted messages for richer formatting:

```yaml
with:
  message: "Alert: High CPU usage detected on server-01"
  formatted_message: "<b>Alert:</b> High CPU usage detected on <code>server-01</code>"
```

## Supported Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `message` | string | Yes | The plain text message to send |
| `formatted_message` | string | No | Optional HTML formatted message |
