## Overview

LINE Notify Provider allows you to send alert messages from Keep via [LINE Notify](https://notify-bot.line.me/), a messaging notification service widely used in Japan and Southeast Asia. LINE Notify enables you to send text messages, images, and stickers directly to LINE users or groups.

The LINE Notify provider supports text messages with optional image URLs and stickers, making it a versatile choice for alert notifications in regions where LINE is the dominant messaging platform.

## Connecting with LINE Notify Provider

1. Generate a LINE Notify access token from the [LINE Notify website](https://notify-bot.line.me/my/).
2. Add LINE Notify Provider in Keep with the generated access token.
3. Connect the LINE Notify Provider with Keep.

## Message Format Support

### Text Messages
Send plain text messages up to 1000 characters:

```yaml
with:
  message: "Alert: High CPU usage detected on server-01"
```

### Messages with Images
Include image URLs in your notifications:

```yaml
with:
  message: "Alert: Server overload detected"
  image_thumbnail: "https://example.com/thumbnail.png"
  image_fullsize: "https://example.com/fullsize.png"
```

### Messages with Stickers
Add LINE stickers to make alerts more noticeable:

```yaml
with:
  message: "Critical Alert!"
  sticker_package_id: "446"
  sticker_id: "1988"
```

### Silent Notifications
Send notifications without triggering push notifications:

```yaml
with:
  message: "Maintenance window starting in 10 minutes"
  notification_disabled: true
```

## Supported Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `message` | string | Yes | The message text (max 1000 characters) |
| `image_thumbnail` | string | No | URL of thumbnail image (max 240x240px) |
| `image_fullsize` | string | No | URL of full-size image (max 2048x2048px) |
| `sticker_package_id` | string | No | LINE sticker package ID |
| `sticker_id` | string | No | LINE sticker ID |
| `notification_disabled` | boolean | No | If true, no push notification is sent |

## Example Workflow

```yaml
id: line-notify-example
name: LINE Notify Example
description: Send a test notification to LINE
triggers:
  - type: manual

actions:
  - name: send-line-message
    provider:
      type: line-notify
      config: "{{ secrets.LINE_NOTIFY }}"
    with:
      message: "🚨 Alert: Database connection failed on {{ event.host }}"
      sticker_package_id: "446"
      sticker_id: "1988"
```
