## Overview

The SolarWinds provider enables seamless integration between Keep and SolarWinds Orion. It allows you to receive alerts from SolarWinds to Keep via webhooks and optionally pull active alerts directly from the SolarWinds Information Service (SWIS) REST API.

## Connecting SolarWinds to Keep

### Webhook (Recommended)

To receive alerts from SolarWinds via webhooks, configure an HTTP notification in SolarWinds:

1. In the SolarWinds Orion Web Console, go to **Settings** > **Alerts & Activity** > **Alerts**.
2. Create or edit an alert and navigate to the **Trigger Actions** tab.
3. Click **Add Action** and select **Send an HTTP Request**.
4. Configure the HTTP request:
   - **URL**: `https://api.keephq.dev/alerts/event/solarwinds`
   - **Method**: `POST`
   - **Content Type**: `application/json`
   - **Body**: Include alert variables such as `${AlertID}`, `${AlertName}`, `${NodeName}`, `${Severity}`, `${AlertMessage}`, `${AlertTriggerTime}`, etc.
5. Save the alert configuration.

### API Pull (Optional)

To pull existing alerts from SolarWinds via the SWIS REST API:

1. Ensure your SolarWinds Orion server has the SWIS REST API enabled (default port `17778`).
2. Configure the provider with your SolarWinds host URL, username, and password.
3. Keep will query `Orion.AlertActive` to fetch active alerts.

## Authentication

| Field      | Required | Description                          |
|------------|----------|--------------------------------------|
| Host URL   | Yes      | SolarWinds Orion server URL          |
| Username   | Yes      | SolarWinds account username          |
| Password   | Yes      | SolarWinds account password          |

## Severity Mapping

SolarWinds severities are mapped to Keep severities as follows:

| SolarWinds Severity | Keep Severity |
|---------------------|---------------|
| CRITICAL            | CRITICAL      |
| WARNING             | WARNING       |
| OK                  | INFO          |
| INFORMATIONAL       | INFO          |
| UNKNOWN             | INFO          |

## Status Mapping

SolarWinds statuses are mapped to Keep statuses as follows:

| SolarWinds Status   | Keep Status   |
|---------------------|---------------|
| ACTIVE              | FIRING        |
| DOWN / CRITICAL     | FIRING        |
| ACKNOWLEDGED        | ACKNOWLEDGED  |
| RESOLVED / UP / OK  | RESOLVED      |
