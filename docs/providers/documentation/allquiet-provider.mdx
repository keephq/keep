## Connecting with the Provider

1. Log into your [AllQuiet](https://app.allquiet.app/) account.
2. Navigate to **Settings > Team > API keys**.
3. Click **Generate API key** and copy the key.
4. Paste the key into the `api_key` field when connecting AllQuiet in Keep.

## What Keep Monitors

Keep polls all **open** and **acknowledged** incidents from AllQuiet and surfaces them alongside alerts from other tools.  Resolved and closed incidents are not ingested by default.
