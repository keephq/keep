## Connecting with the Provider

1. Log into your [Middleware.io](https://app.middleware.io/) account.
2. Navigate to **Settings > API Keys**.
3. Generate a new API key and copy it — this is your `api_key`.
4. Find your **Account ID** in your account settings (typically shown in the URL or profile page).
5. Use both values when configuring the Middleware.io provider in Keep.

## Webhook Integration

To receive alerts in real time, configure a Middleware.io alert rule to send webhook notifications to your Keep ingest endpoint.

1. In Middleware.io, navigate to **Alerts > Alert Rules**.
2. Create or edit a rule and set the notification channel to **Webhook**.
3. Enter your Keep webhook URL as the destination.
