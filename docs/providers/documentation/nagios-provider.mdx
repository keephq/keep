## Connecting with the Provider

1. Create or copy a Nagios XI API key from your Nagios XI user settings.
2. Configure the provider with your Nagios XI base URL, for example `https://nagios.example.com`.
3. The provider polls the Nagios XI REST API for host and service status objects.

## Supported Scope

- Nagios XI REST API using API key authentication.
- Host status polling from `/nagiosxi/api/v1/objects/hoststatus`.
- Service status polling from `/nagiosxi/api/v1/objects/servicestatus`.
