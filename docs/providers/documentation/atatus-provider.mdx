## Connecting with the Provider

1. Log into your [Atatus](https://app.atatus.com/) account.
2. Navigate to **Settings > API Keys**.
3. Copy your API key — this is your `api_key`.
4. Optionally, copy the **Project Token** from your project settings as `project_token` to scope requests to a specific project.

## Webhook Integration

To receive Atatus error alerts in real time:

1. In Atatus, navigate to **Project Settings > Integrations > Webhook**.
2. Add a new webhook and set the URL to your Keep ingest endpoint.
3. Configure the alert conditions you want to forward (e.g., new errors, error spikes).
