## Connecting with the Provider

1. Open WeCom (企业微信) and navigate to the group chat where you want to receive alerts.
2. Click on the group name → "Group Robot" (群机器人) → "Add Robot" (添加机器人).
3. Configure the robot name and click "Add".
4. Copy the **Webhook URL** from the robot configuration page.
