## Overview

WeCom Provider allows you to send alert messages from Keep via [WeCom](https://work.weixin.qq.com) group robot webhooks. WeCom (企业微信) is a team messaging and collaboration platform by Tencent. The provider sends text or markdown messages to a WeCom group using a webhook key.

## Connecting with WeCom Provider

1. Open WeCom and navigate to the group where you want to receive alerts.
2. Click the **Group Settings** (群设置) icon in the top-right corner.
3. Go to **Add Group Robot** (添加群机器人) and choose **New Robot** (新创建一个机器人).
4. Give the robot a name and click **Save**.
5. Copy the generated **Webhook Key** from the URL (the part after `key=`).
6. Add the WeCom Provider in Keep with the copied webhook key.
7. Connect the WeCom Provider with Keep.

## Message Format Support

### Text Messages

Send plain text messages to a WeCom group:

```yaml
with:
  message: "Alert: High CPU usage detected on server-01"
```

### Markdown Messages

Send markdown-formatted messages to a WeCom group:

```yaml
with:
  message: "**Alert:** High CPU usage detected on `server-01`"
  markdown: true
```

## Supported Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `message` | string | Yes | The message text to send |
| `markdown` | boolean | No | Send the message as markdown (default: false) |
| `mentioned_mobile_list` | list | No | List of mobile numbers to @mention |

## Example Workflow

```yaml
id: wecom-example
name: WeCom Example
description: Send a test notification to WeCom
triggers:
  - type: manual

actions:
  - name: send-wecom-message
    provider:
      type: wecom
      config: "{{ secrets.WECOM }}"
    with:
      message: "🚨 Alert: Database connection failed on {{ event.host }}"
```
