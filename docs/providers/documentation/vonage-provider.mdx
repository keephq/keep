## Connecting with the Provider

You'll need a Vonage API key and API secret. Both are available in your [Vonage dashboard](https://dashboard.nexmo.com/).

1. Log in to the [Vonage API Dashboard](https://dashboard.nexmo.com/).
2. Copy the **API key** and **API secret** shown at the top of the page.
3. Set a **From** value — either a virtual number you own or an alphanumeric sender ID (subject to country regulations).
4. Provide the **To** number in E.164 format (e.g. `15551234567`).
