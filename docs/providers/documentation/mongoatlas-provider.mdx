## Connecting with the Provider

In order to connect to MongoDB Atlas and pull alerts, you need to provide your API credentials. You can generate these in the MongoDB Atlas console under Access Management.

Provide the following authentication parameters:
- `public_key`: Your MongoDB Atlas Public API Key.
- `private_key`: Your MongoDB Atlas Private API Key (requires Project Owner or higher permissions for alert access).
- `group_id`: Your MongoDB Atlas Project (Group) ID.

## Notes

- The provider uses MongoDB Atlas API v1.0.
- It maps Atlas severities (`INFORMATIONAL`, `WARNING`, `CRITICAL`) to Keep severities.
- It uses HTTP Digest Authentication to communicate with the Atlas API.
