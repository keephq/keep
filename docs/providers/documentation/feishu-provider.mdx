## Overview

Feishu Provider allows you to send alert messages from Keep via [Feishu](https://www.feishu.cn/) (also known as Lark), a collaboration platform widely used in China and globally. Feishu custom bots enable you to send text messages directly to groups or individual chats through webhook integration.

The Feishu provider supports plain text messages, making it a straightforward choice for alert notifications in teams that use Feishu/Lark as their primary communication platform.

## Connecting with Feishu Provider

1. Create a custom bot in your Feishu group:
   - Open a Feishu group chat
   - Go to **Group Settings** → **Bots** → **Add Bot** → **Custom Bot**
   - Give your bot a name and copy the **Webhook URL** (the token is the last part of the URL)
2. Add Feishu Provider in Keep with the webhook token.
3. Connect the Feishu Provider with Keep.

## Message Format Support

### Text Messages
Send plain text messages:

```yaml
with:
  message: "Alert: High CPU usage detected on server-01"
```

## Supported Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `message` | string | Yes | The message text to send |

## Example Workflow

```yaml
id: feishu-example
name: Feishu Example
description: Send a test notification to Feishu
triggers:
  - type: manual

actions:
  - name: send-feishu-message
    provider:
      type: feishu
      config: "{{ secrets.FEISHU }}"
    with:
      message: "🚨 Alert: Database connection failed on {{ event.host }}"
```
