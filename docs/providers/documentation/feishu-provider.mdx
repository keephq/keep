## Connecting with the Provider

1. Open the Feishu/Lark group chat where you want to receive alerts.
2. Click "Settings" → "Bots" → "Add Bot" → "Custom Bot".
3. Configure the bot name and (optionally) enable signature verification.
4. Copy the **Webhook URL** and optionally the **Signing Key**.
