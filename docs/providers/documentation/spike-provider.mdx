## Connecting with the Provider

Spike.sh integrates with Keep through a generic inbound webhook.

1. In Spike.sh, go to **Services** → select or create a service.
2. Click **Integrations** → **Generic Webhook**.
3. Copy the webhook URL (starts with `https://hooks.spike.sh/...`).
4. Paste it as the **Webhook URL** when adding this provider in Keep.

## Alert Statuses

| Keep status value | Effect in Spike.sh |
|---|---|
| `critical` | Opens a new incident (high severity) |
| `warning` | Opens a new incident (warning severity) |
| `info` | Informational alert, no on-call page |
| `resolved` | Resolves an open incident |
