## Connecting with the Provider

To connect Keep to FireHydrant, you need a Bot API Token:

1. Log in to your [FireHydrant](https://app.firehydrant.io) account.
2. Navigate to **Settings** → **Bot Users**.
3. Click **Create Bot User** and give it a descriptive name (e.g., "Keep Integration").
4. Copy the generated API token.
5. Enter the token as `api_key` when configuring the FireHydrant provider in Keep.

## Webhook Integration

To receive real-time incident notifications in Keep:

1. In FireHydrant, navigate to **Settings** → **Integrations** → **Webhooks**.
2. Click **Add Webhook**.
3. Enter your Keep webhook URL: `https://<your-keep-url>/alerts/event/firehydrant`.
4. Select the incident events to forward: **Incident Created**, **Milestone Updated**, **Incident Resolved**, etc.
5. Click **Save**.

Keep will automatically create, update, and resolve alerts as FireHydrant incidents progress through their lifecycle milestones.
