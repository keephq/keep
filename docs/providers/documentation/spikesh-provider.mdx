## Connecting with the Provider

1. Log into your [Spike.sh](https://app.spike.sh/) account.
2. Navigate to **Settings > API Tokens**.
3. Generate a new API token and copy it — this is your `api_token`.
4. Optionally, create an integration in Spike.sh and copy its **webhook token** as `integration_token` to enable sending alerts from Keep to Spike.sh.

## Webhook Integration

To receive Spike.sh incidents in Keep as alerts in real time:

1. In Spike.sh, navigate to **Integrations** and add a new integration of type **Webhook**.
2. Set the webhook URL to your Keep ingest endpoint.
3. Configure alert routing rules in Spike.sh to send relevant incidents to this integration.
