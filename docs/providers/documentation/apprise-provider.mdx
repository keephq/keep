## Connecting with the Provider

1. Deploy [Apprise API](https://github.com/caronc/apprise-api) server (Docker recommended):
   ```bash
   docker run -p 8000:8000 caronc/apprise-api:latest
   ```
2. Configure your notification services (Slack, Discord, email, etc.) in Apprise via the web UI or config files.
3. Use the server URL and tag in Keep when adding the provider.
