## Connecting with the Provider

To connect Keep to Rootly, you need a Rootly API key:

1. Log in to your [Rootly](https://rootly.com) account.
2. Navigate to **Settings** → **API Keys**.
3. Click **New API Key**, give it a descriptive name (e.g., "Keep Integration"), and click **Create**.
4. Copy the generated API key — you will not be able to view it again.
5. Use this key as the `api_key` parameter when configuring the Rootly provider in Keep.

## Webhook Integration

To receive real-time incident updates in Keep:

1. In Rootly, navigate to **Settings** → **Integrations** → **Webhooks**.
2. Click **Add Webhook**.
3. Set the **URL** to your Keep webhook endpoint: `https://<your-keep-url>/alerts/event/rootly`.
4. Select the events you want to forward (e.g., incident created, updated, resolved).
5. Save the webhook.

Keep will automatically parse incoming Rootly webhook payloads and create or update alerts accordingly.
