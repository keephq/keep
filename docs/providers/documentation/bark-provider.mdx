## Getting your Device Key

1. Install the [Bark app](https://apps.apple.com/app/bark-customed-notifications/id1403753865) on your iPhone.
2. Open the app — your device key is displayed on the main screen (e.g. `https://api.day.app/your_device_key`).
3. Copy the key portion (the path segment after `api.day.app/`).

## Self-hosted Bark server

If you run your own Bark server, set `server_url` to your instance URL (e.g. `https://bark.example.com`).
Leave it blank to use the default cloud endpoint (`https://api.day.app`).
