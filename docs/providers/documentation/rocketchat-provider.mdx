## Connecting with the Provider

To connect with the Rocket.Chat provider and send messages through Keep, follow these steps:

1. **Obtain a Rocket.Chat Personal Access Token:**
   - Log in to your Rocket.Chat server.
   - Go to your profile settings.
   - Navigate to **Personal Access Tokens**.
   - Generate a new token and copy the token value and your User ID.

2. **Get your Rocket.Chat User ID:**
   - The User ID is displayed alongside your Personal Access Token in the profile settings.

3. **Configure the Rocket.Chat provider in Keep** with:
   - `access_token`: Your Personal Access Token
   - `user_id`: Your User ID
   - `base_url`: Your Rocket.Chat server URL (e.g., `https://chat.example.com`)

4. Use the following YAML example to send a message using the Rocket.Chat provider:

```yaml title=examples/alert_example.yml
# Send a message to a Rocket.Chat channel.
alert:
  id: rocketchat-notification
  description: Send a notification to Rocket.Chat
  actions:
    - name: send-to-rocketchat
      provider:
        type: rocketchat
        config: "{{ providers.rocketchat-provider }}"
        with:
          message: "Alert triggered!"
          room: "#general"
```
