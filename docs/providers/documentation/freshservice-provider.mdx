## Overview

The Freshservice Provider integrates Keep with [Freshservice](https://freshservice.com/), Freshworks' cloud-based IT service management (ITSM) platform. This integration enables you to:

- **Pull tickets and incidents** from Freshservice into Keep as alerts
- **Create new tickets** in Freshservice from Keep workflows
- Receive real-time updates via **webhooks** when ticket status or priority changes

## Connecting with the Provider

1. Log in to your Freshservice account.
2. Navigate to **Profile Settings** (click your avatar in the top-right corner).
3. Scroll down to find the **API Key** section and copy your API key.
4. Note your Freshservice domain (e.g. `yourcompany.freshservice.com`).
5. In Keep, create a new Freshservice provider with your **API Key** and **Domain**.

## Webhook Setup

To receive real-time ticket updates from Freshservice:

1. In Freshservice, go to **Admin** → **Workflow Automator**.
2. Create a new automation triggered by ticket events (e.g. "Ticket is Created", "Ticket is Updated").
3. Add an action **Trigger Webhook** and set:
   - **Callback URL**: `{keep_webhook_api_url}`
   - **Request type**: POST
   - **Encoding**: JSON
   - Add a custom header `x-api-key` with your Keep API key
4. Select the ticket fields you want to include in the payload.
5. Save and activate the automation.
