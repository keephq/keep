## Connecting with the Provider

1. In Zulip, go to **Settings → Your bots → Add a new bot**.
2. Create an **Incoming webhook** or **Generic bot**.
3. Copy the **bot email** and **API key**.
4. Note your Zulip server URL (e.g. `https://yourorg.zulipchat.com`).
