## Connecting with the Provider

1. Open the DingTalk group chat where you want to receive alerts.
2. Click on the group name → "Group Settings" → "Intelligent Group Assistant" → "Add Robot".
3. Choose "Custom" robot type and configure a name.
4. Select security settings: optionally enable "Sign" for HMAC verification.
5. Copy the **Webhook URL** and optionally the **Secret**.
