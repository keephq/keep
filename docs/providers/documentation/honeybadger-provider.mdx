## Connecting with the Provider

To connect Keep to Honeybadger, you need a Personal Auth Token and your Project ID:

1. Log in to your [Honeybadger](https://app.honeybadger.io) account.
2. Navigate to **Profile** → **Authentication Token** to find your personal API token.
3. Find your **Project ID** in the URL when viewing a project: `app.honeybadger.io/projects/<project_id>`.
4. Use the token as `api_key` and the project ID as `project_id` when configuring the provider in Keep.

## Webhook Integration

To receive real-time error notifications in Keep:

1. In Honeybadger, open your project and navigate to **Settings** → **Webhooks**.
2. Click **Add Webhook**.
3. Enter your Keep webhook URL: `https://<your-keep-url>/alerts/event/honeybadger`.
4. Select the events to forward: **Error Occurred**, **Error Resolved**, **Error Ignored**.
5. Click **Save**.

Keep will automatically create and resolve alerts when Honeybadger fires webhooks.
