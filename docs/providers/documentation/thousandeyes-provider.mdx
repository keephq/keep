## Authentication Parameters

The `api_token` is required for connecting to the ThousandEyes provider. You can obtain it as described in the "Connecting with the Provider" section.

## Connecting with the Provider

### API Token

To obtain the ThousandEyes API token, follow these steps:

1.  Log in to your ThousandEyes account.
2.  Navigate to "Settings" -> "User Settings".
3.  Scroll down to the "API Keys" section.
4.  Click "Generate API Key".
5.  Copy the generated API token and keep it for further use in Keep.

## Scopes

-   `read`
    | Required: `True`
    | Description: Allows reading data from the ThousandEyes API, including alerts, tests, and network performance metrics.

## Notes

<Note>
    The ThousandEyes provider allows you to monitor network performance, troubleshoot issues, and receive alerts based on predefined thresholds and conditions.
</Note>
