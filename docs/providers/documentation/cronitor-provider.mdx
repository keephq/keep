## Overview

[Cronitor](https://cronitor.io/) is a monitoring platform for cron jobs, background workers, and scheduled tasks. It also monitors websites and APIs. When a job fails to run on time, runs too long, or a site goes down, Cronitor pages your team.

The Keep Cronitor provider lets you:
- Pull failing monitors from the Cronitor API into Keep
- Receive real-time webhook notifications when jobs miss their schedule, fail, or recover
