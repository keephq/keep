## Overview

The UptimeRobot Provider integrates Keep with [UptimeRobot](https://uptimerobot.com/), a popular cloud-based uptime monitoring service. This integration enables you to:

- **Pull monitor statuses** from UptimeRobot into Keep as alerts
- Receive **real-time downtime and recovery notifications** via webhooks
- Track uptime ratios, response times, and SSL certificate expiry

## Connecting with the Provider

1. Log in to your [UptimeRobot Dashboard](https://dashboard.uptimerobot.com/).
2. Go to **My Settings** → **API Settings**.
3. Copy your **Main API Key** (or create a monitor-specific API key).
4. In Keep, create a new UptimeRobot provider using this API key.

## Webhook Setup

To receive real-time alerts from UptimeRobot in Keep:

1. In UptimeRobot, go to **My Settings** → **Alert Contacts**.
2. Click **Add Alert Contact**.
3. Select **Webhook** as the alert contact type.
4. Set the **URL to Notify** to `{keep_webhook_api_url}`.
5. Add a custom HTTP header: `x-api-key: <your Keep API key>`.
6. Save the alert contact.
7. Assign this alert contact to your monitors by editing each monitor.
