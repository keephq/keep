## Connecting with the Provider

1. Go to [Cisco Webex Developer Portal](https://developer.webex.com/my-apps) and create a new Bot.
2. Copy the **Bot Access Token**.
3. Add the bot to the Webex space/room where you want alerts.
4. Get the **Room ID** by calling the [Webex Rooms API](https://developer.webex.com/docs/api/v1/rooms/list-rooms).
