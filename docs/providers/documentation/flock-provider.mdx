## Connecting with the Provider

1. In Flock, go to **Apps** and search for **Incoming Webhooks**.
2. Select the channel you want to post alerts to and create a new webhook.
3. Copy the generated **Webhook URL** (format: `https://api.flock.com/hooks/sendMessage/...`).
4. Paste the URL into the **Webhook URL** field when configuring the Flock provider in Keep.
