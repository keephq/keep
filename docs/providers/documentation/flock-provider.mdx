## Overview

Flock Provider allows you to send alert messages from Keep via [Flock](https://flock.com) incoming webhooks. Flock is a team messaging and collaboration platform. The provider sends plain text messages to a Flock channel using a webhook URL.

<AutoGeneratedSnippet />

## Connecting with Flock Provider

1. Open the [Flock Developer Dashboard](https://dev.flock.com) and sign in.
2. Click **Webhooks** on the left and choose **Add Incoming Webhook**.
3. Select the channel where messages should be posted and give the webhook a name.
4. Click **Save and Generate URL** and copy the generated webhook URL.
5. Add the Flock Provider in Keep with the copied webhook URL.
6. Connect the Flock Provider with Keep.

## Message Format Support

### Text Messages

Send plain text messages to a Flock channel:

```yaml
with:
  message: "Alert: High CPU usage detected on server-01"
```

## Supported Parameters

| Parameter | Type   | Required | Description                      |
|-----------|--------|----------|----------------------------------|
| `message` | string | Yes      | The message text to send         |

## Example Workflow

```yaml
id: flock-example
name: Flock Example
description: Send a test notification to Flock
triggers:
  - type: manual

actions:
  - name: send-flock-message
    provider:
      type: flock
      config: "{{ secrets.FLOCK }}"
    with:
      message: "🚨 Alert: Database connection failed on {{ event.host }}"
```
