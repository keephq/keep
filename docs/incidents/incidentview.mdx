title: "Incident View"
---

The Incident View in Keep provides a detailed interface for managing and tracking individual incidents. This comprehensive view combines all relevant information, actions, and collaboration tools needed for effective incident management.

## Interface Components

### Header Section
- **Incident ID**: Unique identifier
- **Title**: Descriptive incident name
- **Status**: Current incident state
- **Priority**: Incident priority level
- **Owner**: Assigned incident owner
- **Created**: Incident creation time
- **Duration**: Time since creation

### Main Sections

#### Overview Panel
- **Description**: Detailed incident description
- **Impact**: Affected systems and users
- **Timeline**: Chronological event tracking
- **Tags**: Incident categorization

#### Alert Panel
- **Related Alerts**: Connected alert list
- **Alert Timeline**: Alert occurrence timeline
- **Alert Metrics**: Key alert statistics
- **Add/Remove Alerts**: Alert management

#### Communication Panel
- **Comments**: Team discussions
- **Updates**: Status changes
- **Notifications**: Alert configurations
- **Attachments**: Related files

## Key Features

### Incident Management
- **Status Updates**: Change incident state
- **Priority Adjustment**: Modify incident priority
- **Team Assignment**: Assign and reassign ownership
- **Timeline Tracking**: Record all actions and changes

### Alert Integration
- **Alert Association**: Link relevant alerts
- **Alert Correlation**: View related alerts
- **Bulk Actions**: Manage multiple alerts
- **Alert Filters**: Focus on specific alerts

### Collaboration Tools
- **Comment System**: Team communication
- **File Sharing**: Document attachment
- **Notification Rules**: Alert configurations
- **Integration**: External tool connections

## Actions Available

### Primary Actions
1. **Update Status**: Change incident state
2. **Assign Owner**: Transfer ownership
3. **Add Comment**: Post updates
4. **Link Alerts**: Associate alerts

### Secondary Actions
1. **Export Data**: Download incident details
2. **Share View**: Collaborate with others
3. **Configure Notifications**: Set up alerts
4. **Generate Report**: Create incident summary

## Keyboard Shortcuts

- **Ctrl/Cmd + S**: Save changes
- **Ctrl/Cmd + E**: Edit mode
- **Ctrl/Cmd + K**: Quick actions
- **Esc**: Cancel/Close
