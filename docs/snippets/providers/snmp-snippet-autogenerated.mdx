- **listen_port**: UDP port to listen for SNMP traps on (required: False, sensitive: False)
- **community_string**: SNMPv1/v2c community string used to authenticate incoming traps (required: False, sensitive: True)
- **snmp_version**: SNMP protocol version to accept (v1 or v2c) (required: False, sensitive: False)
