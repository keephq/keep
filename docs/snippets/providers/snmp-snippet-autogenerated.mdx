## Authentication
This provider requires authentication.
- **host**: Listen address (required: True, sensitive: False)
- **port**: UDP port for SNMP traps (required: True, sensitive: False)
- **community**: SNMP community string (v1/v2c) (required: True, sensitive: False)

Certain scopes may be required to perform specific actions or queries via the provider. Below is a summary of relevant scopes and their use cases:
- **listen_udp**: Bind to the configured UDP address and receive SNMP notifications. (mandatory)
