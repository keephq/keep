## Authentication
This provider requires authentication.
- **deployment_url**: Cribl instance base URL (required: True)
- **api_token**: Cribl API token (Bearer) (required: False, sensitive: True)
- **username**: Cribl admin username (required: False)
- **password**: Cribl admin password (required: False, sensitive: True)
- **verify_ssl**: Verify TLS/SSL certificates (required: False)
