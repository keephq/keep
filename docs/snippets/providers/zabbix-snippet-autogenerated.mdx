## Authentication
This provider requires authentication.
- **zabbix_frontend_url**: Zabbix Frontend URL (required: True, sensitive: False)
- **auth_token**: Zabbix Auth Token (required: True, sensitive: True)

Certain scopes may be required to perform specific actions or queries via the provider. Below is a summary of relevant scopes and their use cases:
- **problem.get**: The method allows to retrieve problems. (mandatory) ([Documentation](https://www.zabbix.com/documentation/current/en/manual/api/reference/problem/get))
- **script.get**: The method allows to retrieve media types.  ([Documentation](https://www.zabbix.com/documentation/current/en/manual/api/reference/mediatype/get))
- **script.update**: This method allows to update existing media types.  ([Documentation](https://www.zabbix.com/documentation/current/en/manual/api/reference/mediatype/update))
- **script.create**: This method allows to create new media types.  ([Documentation](https://www.zabbix.com/documentation/current/en/manual/api/reference/mediatype/create))
- **event.acknowledge**: This method allows to update events.  ([Documentation](https://www.zabbix.com/documentation/current/en/manual/api/reference/event/acknowledge))
