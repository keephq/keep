## Authentication
This provider requires authentication.
- **region**: AWS region where Bedrock is available (e.g. us-east-1) (required: True, sensitive: False)
- **access_key**: AWS access key ID (leave empty to use IAM role / instance profile) (required: False, sensitive: True)
- **secret_access_key**: AWS secret access key (leave empty to use IAM role / instance profile) (required: False, sensitive: True)
- **session_token**: AWS session token (required only for temporary credentials) (required: False, sensitive: True)
