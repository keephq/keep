## Authentication
This provider requires authentication.
- **device_key**: Bark device key from the Bark app (required: True, sensitive: True)
- **server_url**: Bark server URL (default: https://api.day.app, or your self-hosted URL) (required: False, sensitive: False)


## In workflows

This provider can be used in workflows.



As "action" to make changes or update data, example:
```yaml
actions:
    - name: Query bark
      provider: bark
      config: "{{ provider.my_provider_name }}"
      with:
        message: {value}  # The body of the notification.
        title: {value}  # The notification title.
        level: {value}  # Notification interruption level: active, timeSensitive, passive, or critical.
```
