## Authentication

This provider requires authentication.

- **host_url**: SolarWinds Orion Host URL (required: True, sensitive: False)
- **username**: SolarWinds Orion Username (required: True, sensitive: False)
- **password**: SolarWinds Orion Password (required: True, sensitive: True)
- **verify_ssl**: Verify SSL certificate (required: False, sensitive: False)

Certain scopes may be required to perform specific actions or queries via the provider. Below is a summary of relevant scopes and their use cases:

- **read_alerts**: Read active alerts from SolarWinds Orion via SWIS API
