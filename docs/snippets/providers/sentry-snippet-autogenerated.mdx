## Authentication
This provider requires authentication.
- **api_key**: Sentry Api Key (required: True, sensitive: True)
- **organization_slug**: Sentry organization slug (required: True, sensitive: False)
- **api_url**: Sentry API URL (required: False, sensitive: False)
- **project_slug**: Sentry project slug within the organization (required: False, sensitive: False)
