## How to access PRTG Network Monitor

PRTG is a commercial network monitoring tool. You can try it with a free trial:

1. Download from https://www.paessler.com/prtg
2. Install and start PRTG
3. Log in at `https://localhost` (default: prtgadmin / prtgadmin)

Get your **passhash** from: Setup > My Account > Account Settings > Passhash

The PRTG JSON API is available at `https://your-prtg-server/api/`.
Reference: https://www.paessler.com/manuals/prtg/application_programming_interface_api_definition
